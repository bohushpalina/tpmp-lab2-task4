## 1. Цель работы

Проверка корректности работы функции `process_cars()`, которая:

* считывает данные из входного файла;
* выводит все записи в консоль;
* записывает в выходной файл автомобили старше 2 лет.

Условие фильтрации:

```
(current_year - year_made) > 2
```

Тестируемое значение:

```
current_year = 2026
```

---

## 2. Основной тест

### Входной файл `cars.txt`

```
Toyota Red SN123 AA1111 2020 2025 15000
BMW Black SN456 BB2222 2025 2026 30000
Audi White SN789 CC3333 2023 2026 25000
Ford Blue SN321 DD4444 2022 2025 18000
```

### Контрольный расчёт

| Марка  | Год  | 2026 - год | Старше 2 лет | Попадает в output |
| ------ | ---- | ---------- | ------------ | ----------------- |
| Toyota | 2020 | 6          | Да           | Да                |
| BMW    | 2025 | 1          | Нет          | Нет               |
| Audi   | 2023 | 3          | Да           | Да                |
| Ford   | 2022 | 4          | Да           | Да                |

### Ожидаемый `filtered_cars.txt`

```
Toyota Red SN123 AA1111 2020 2025 15000.00
Audi White SN789 CC3333 2023 2026 25000.00
Ford Blue SN321 DD4444 2022 2025 18000.00
```
### Реальный вывод `filtered_cars.txt`

```
Toyota Red SN123 AA1111 2020 2025 15000.00
Audi White SN789 CC3333 2023 2026 25000.00
Ford Blue SN321 DD4444 2022 2025 18000.00
```

Результат корректный

## 3. Граничный тест

Запись:

```
Kia Green SN999 EE5555 2024 2026 17000
```

```
2026 - 2024 = 2
```

Условие `> 2` → автомобиль **не попадает** в файл.

Результат корректный.
